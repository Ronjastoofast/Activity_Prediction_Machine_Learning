The workflow for this code is as follows:

(optional workflow: start from the beginning. the code and files for these steps are in iteration1_code:
EOS_data.csv is the raw data

generate_IC50.py uses EOS_data.csv to generate ic50_results.csv

generate_molecular_descriptors.py uses ic50_results.csv and generates descriptors.csv)

STARTING from an edited version the existing descriptors.csv, with the untrustworthy IC50 result removed
1. feature_selector.py uses descriptors.py, and generates a cutdown list of features to use in Test_different_ML_models.py

2. copy these lists into train_and_save_XGBoost_model.py (this has already been done here)

3. train_and_save_XGBoost_model uses descriptors.csv to train and save a ML model

5. plot_feature_importance uses the model and descriptors.csv and plots the importance of different feature columns

6. find_molecules_by_ecfp4_bit uses descriptors.csv and the number of an ecfp4 bit (for example, ecfp4_70), generates compounds_with_ecfp_X.csv
